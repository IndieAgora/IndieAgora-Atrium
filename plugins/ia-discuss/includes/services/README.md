# includes / services

Longer-lived service classes that encapsulate auth, phpBB access, notifications, uploads, and membership logic.

## File tree
```text
├── auth.php
├── membership.php
├── notify.php
├── phpbb-write.php
├── phpbb.php
├── text.php
└── upload.php
```

## File roles
- `auth.php` — PHP source file used by the plugin runtime.
- `membership.php` — PHP source file used by the plugin runtime.
- `notify.php` — PHP source file used by the plugin runtime.
- `phpbb-write.php` — PHP source file used by the plugin runtime.
- `phpbb.php` — PHP source file used by the plugin runtime.
- `text.php` — PHP source file used by the plugin runtime.
- `upload.php` — PHP source file used by the plugin runtime.
