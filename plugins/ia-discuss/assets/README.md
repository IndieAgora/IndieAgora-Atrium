# assets

Front-end asset container for styles, scripts, and static images used by the Discuss panel.

## File tree
```text
├── css/
├── img/
└── js/
```

## File roles
- `css/` — Runtime stylesheets. Base, layout, modal, topic, moderation, audio, and feature-specific CSS live here.
- `img/` — Static image assets referenced by the UI.
- `js/` — Runtime JavaScript bundles and already-modular UI files that are loaded by WordPress.

## Maintenance entry point
Use this folder README together with the sibling NOTES file before changing anything in the folder.
